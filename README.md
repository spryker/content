# Content Module
[![Build Status](https://travis-ci.org/spryker/content.svg)](https://travis-ci.org/spryker/content)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%207.2-8892BF.svg)](https://php.net/)

## Installation

